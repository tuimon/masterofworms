# Master of Worms

Always wanted to combine Pacman and Snake? And play it on terminal? 

Now you can.

## Install

Compile: 
```gcc maze.c -lncurses -o maze```
Run: 
```./maze```

Note: You need ncurses. 

## Gameplay

keys: a,s,d,w to control the snekman. Try to get as big as you can. 120 seconds to go.
